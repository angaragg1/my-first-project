# Төслийн эхний даалгавар
Энэ даалгаварт бүтээгдэхүүний жагсаалтыг харуулах юм.
Дараах 4 кодын аль нэгийн ажиллуулж хөгжүүлэлтийн орчинд ажиллуулна.
```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```
Дараа нь суурин компьютер дээрээ http://localhost:3000-г дурын веб хөтчөд дээрээ харна.
Хөгжүүлэгдсэн хуудсыг харахыг хүсвэл дараах линкээр орно уу: https://my-first-project-ft8s.vercel.app
